Meus Projetos

Projeto 1: cadastro de um pet shop

Este projeto é um sistema de cadastro de animais de estimação desenvolvido para um pet shop. O objetivo é simplificar o processo de registro de pets e suas informações, como nome, idade, raça e proprietário. Utilizando HTML e CSS, a interface é intuitiva e amigável, permitindo que os usuários acessem facilmente as funcionalidades. O sistema é responsivo, garantindo uma experiência agradável em dispositivos móveis e desktops. Com essa ferramenta, o pet shop consegue gerenciar de forma eficaz os dados dos animais, proporcionando um atendimento de qualidade aos clientes.

projeto 2: site de venda de celular

Este projeto consiste em uma aplicação web desenvolvida para promover um produto inovador nas redes sociais. O site foi criado com HTML e CSS, focando em uma apresentação visual atrativa e de fácil navegação. Ele destaca as principais características do produto, além de incluir informações relevantes que auxiliam na decisão de compra. O design é otimizado para dispositivos móveis, permitindo que os usuários acessem o conteúdo de maneira fluida em qualquer plataforma. A aplicação busca não apenas informar, mas também engajar os visitantes, incentivando a interação e a conversão.
